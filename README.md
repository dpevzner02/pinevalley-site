## Local Preview

Open `index.html` in a browser.

## Fastest Public Hosting: Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag the entire `pinevalley-site` folder onto the page.
3. Netlify will create a temporary public URL.
4. Test the buttons.
5. If it looks good, connect `pinevalley.ai` later.

## Permanent Free Hosting: GitHub Pages

1. Create a GitHub account or sign in at https://github.com
2. Create a new public repository named `pinevalley-site`.
3. Upload these files and folders:
   - `index.html`
   - `assets/`
   - `README.md`
4. In GitHub, go to Settings -> Pages.
5. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. GitHub will give you a public URL like:
   `https://YOUR-USERNAME.github.io/pinevalley-site/`

## Custom Domain Later

After the site is published and tested, point `pinevalley.ai` DNS to the selected host. Keep Zoho Mail MX/TXT records unchanged.
